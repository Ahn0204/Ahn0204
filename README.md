# 🔥 Problem Solving

## Spring Security CSRF 403 문제 해결

AJAX POST 요청 과정에서 발생한  
403 Forbidden 문제를 분석하며  
CSRF 토큰 전달 구조와 요청 흐름을 이해하고 해결했습니다.

---

## JPA N+1 문제 분석 및 개선

주문 목록 조회 과정에서 발생한  
N+1 문제를 분석하고 Fetch Join을 적용하여  
불필요한 쿼리 발생을 줄이며 조회 성능을 개선했습니다.
