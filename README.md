# 🔥 Problem Solving Experience

### 주문 데이터 조회 성능 개선
주문 목록 조회 과정에서 연관 데이터가 반복 조회되며 성능 저하 문제가 발생했습니다.

문제 원인을 분석하며 JPA 연관관계와 조회 구조를 개선했고,  
Fetch Join을 적용하여 불필요한 쿼리 발생을 줄이며 조회 성능을 개선했습니다.

---

### Spring Security CSRF 문제 해결
AJAX POST 요청 과정에서 403 Forbidden 오류가 발생했습니다.

원인을 분석한 결과 CSRF 토큰이 요청 Header에 포함되지 않은 문제였고,  
meta 태그와 AJAX Header 설정을 통해 문제를 해결했습니다.
